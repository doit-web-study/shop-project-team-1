# Shop

> 간단한 쇼핑몰을 만들어봅시다!

### 프로젝트 목적
- 쇼핑몰을 만들면서 기본적인 웹 개발을 익히기 위함
- RESTful API를 이용한 CRUD 구현 방법을 익히기 위함
- Spring 프레임워크 동작 방식과 JPA 사용 방법을 배우기 위함

### 사용 기술
- Spring Boot 3.3.4
- Spring Data JPA
- Springdoc OpenAPI  2.0.2
